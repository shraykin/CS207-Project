Homemade Vending Machine
By: Jeffrey Li
Date created: December 4, 2019


Description
This code is designed for a vending machine using stepper motors, an LCD display, and a 4x4 keypad.
Each vending slot is activated when the user enters the password onto the keypad. 
The passwords are:
Top Left: 1311
Top Right: 5432
Bottom Left: 6542
Bottom Right: 976A


Parts List
-Hardboard
-Carpenter glue
-Hotglue
-Wires
-Plastic screen(clear)
-1602 LCD display screen
-4x4 Matrix Keypad
-4x 28BYJ-48 stepper motor
-4x ULN2003 driver
-10k potentiometer
-Jumper wires
-Arduino Mega 2560
